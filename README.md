# ATOM 2026 - LIP6, Paris

**Modéliser et résoudre des problèmes d'optimisation multi-objectif avec JuMP et MultiObjectiveAlgorithms** (présentation invitée).

Cet exposé propose une introduction pratique à l’optimisation multi-objectif avec Julia, centrée sur l’utilisation de JuMP et du package MOA. Les participants sont invités à suivre les démonstrations en direct sur leur propre ordinateur. L’objectif est d’aborder rapidement la modélisation, la résolution et l’analyse de problèmes multi-objectifs.
